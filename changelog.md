## 1.1.0

> 发布于 2026-06-30

### 新增

- 新增更新日志板块
- 云端配置可视化编辑，支持对本地缓存的云端 JSON 规则进行可视化编辑
- 可视化编辑支持搜索功能，支持按包名/应用名搜索 App，按 活动 ID 搜索活动
- 订阅列表长按拖拽排序，通过长按拖拽更改规则订阅顺序
- 悬浮窗识别功能，一键识别前台应用的包名、应用名称、活动 ID
- 悬浮窗识别结果可单独点击复制，复制时显示 Toast 提示
- 悬浮窗识别支持一键添加到可视化编辑器（自动填入包名并打开应用/活动编辑面板）
- 屏幕取色功能，通过悬浮窗十字准星实时拾取屏幕颜色并填入取色器
- 「更改后自动应用」设置选项，手动修改规则并确认后自动触发应用（默认开启）

### 优化

- 规则入口和导出按钮移动到同一行，减少界面空间占用
- "Activity" 统一翻译为"活动"
- 优化可视化编辑的三层级界面的展开/关闭动画和逻辑

### 修复

- 修复活动编辑逻辑中的数据传播问题（子层确认后再取消，父层仍保留了变更）
- 修复ci工作流异常的问题

## 1.0.0

> 发布于 2026-06-29

### 新增

- **多云订阅支持** — 添加云端 JSON 订阅源或本地 JSON，自动拉取并解析 `NBIRules`
- **优先级合并** — 多订阅源按优先级智能合并，按包名去重
- **智能格式转换** — 自动检测 HyperOS 版本（OS2.2 / OS3.0 / OS3.3），输出对应格式（XML / JSON）
- **一键应用 & 热重载** — Root 写文件 + `cmd miui_navigation_bar_immersive update` 即时生效
- **还原官方规则** — 支持备份 / 恢复系统默认规则
- **开机自动应用** — 通过 `BootReceiver` 开机后自动应用规则
- **3 种导航栏样式** — 标准 / 悬浮 / 液态玻璃（iOS 风格毛玻璃效果）
- **6 种主题模式** — 系统跟随 / 浅色 / 深色 / Monet 跟随 / Monet 浅色 / Monet 深色
- **中英文双语** — 支持简体中文和英文，跟随系统语言
- **设置导入导出** — 一键备份 / 恢复全部配置
- **自动定时更新** — 订阅源按设定间隔自动拉取更新并合并应用
- **Miuix 设计语言** — 基于 Miuix Compose UI 组件库的 HyperOS 风格界面
- **关于页面** — 液态玻璃毛玻璃效果，显示版本信息、开源许可和项目链接
